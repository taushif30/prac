Q4: Inheritance?


  ANS:Inheritance is a fundamental concept in object-oriented programming that allows objects to inherit properties and methods from other objects. It enables code reuse, promotes modularity, and 
facilitates the creation of hierarchical relationships between objects.
  - In JavaScript, inheritance is typically achieved through prototype-based inheritance. Objects in JavaScript can have a prototype, which is another object from which they inherit properties and methods.


Q5:Prototype?


  ANS:- The prototype is an object that is associated with every JavaScript object. It serves as a blueprint or template for creating new objects through inheritance. 
The prototype object contains properties and methods that are shared among all instances created from it.
