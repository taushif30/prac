Q4 -What is a Temporal Dead Zone?


    ANS:The Temporal Dead Zone (TDZ) is a concept introduced in ECMAScript 6 (ES6) that refers to a specific behavior that occurs when trying to access a variable before it has been 
             declared using the let or const keyword.Return : Refrence Error.
         Benefit : Improve the code quality.
